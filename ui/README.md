# UI 디자인 참고 이미지

이 디렉토리에는 푸름대학교 학생성장지원센터 CHAMP 시스템 개발을 위한 참고 디자인 스크린샷이 포함되어 있습니다.

## 이미지 목록

### 우석대학교 CHAMP 웹사이트 스크린샷 (홈 페이지)

실제 운영 중인 우석대학교 CHAMP 웹사이트(https://champ.woosuk.ac.kr)의 홈 페이지를 캡처한 이미지들입니다.

- **1.png** (1798 x 963px) - 히어로 슬라이더 및 상단 섹션
- **2.png** (1512 x 960px) - 4개 역량 아이콘 그리드
- **3.png** (1533 x 970px) - 프로그램 카드 그리드 (상단)
- **4.png** (1523 x 965px) - 프로그램 카드 그리드 (중단)
- **5.png** (1463 x 957px) - 프로그램 카드 그리드 (하단) 및 전체보기 버튼

## 구현 완료 사항

위 스크린샷을 기반으로 다음 기능들이 구현되었습니다 (커밋: b0c38c6):

### 홈 페이지 (`src/main/resources/templates/index.html`)

1. **히어로 슬라이더**
   - 3개 슬라이드 자동 회전 (5초 간격)
   - 좌우 화살표 네비게이션
   - 핵심역량 진단, 비교과 프로그램, CHAMP 마일리지

2. **4개 역량 아이콘 그리드**
   - 전체보기 📊
   - 문제해결역량 🧩
   - 자기관리역량 🏅
   - 공감소통역량 🏛️

3. **필터 섹션**
   - 행정부서 드롭다운 (📱 전체 행정부서)

4. **8개 프로그램 카드**
   - D-day 뱃지 (D-3, D-4, 마감, 모집완료, D-24, D-35)
   - 즐겨찾기 버튼 ⭐
   - HITS 카운터
   - 부서명
   - 프로그램 제목 및 카테고리
   - 신청/운영 기간
   - 진행률 바
   - 참여 인원 (현재/최대)

5. **전체보기 버튼**
   - /programs 페이지로 이동

6. **반응형 디자인**
   - 호버 효과 및 애니메이션
   - 완전한 반응형 레이아웃

## 사용 방법

### 새 세션에서 디자인 참고하기

새로운 Claude 세션에서 이 이미지들을 참고하려면:

```
Read 도구를 사용하여 이미지를 읽습니다:
- /home/user/SCMS2/ui/1.png
- /home/user/SCMS2/ui/2.png
- /home/user/SCMS2/ui/3.png
- /home/user/SCMS2/ui/4.png
- /home/user/SCMS2/ui/5.png
```

### 추가 개발 시

다른 페이지(프로그램 목록, 마일리지, 상담 등)를 개발할 때도 이 디자인 스타일을 참고하여 일관성 있는 UI/UX를 유지하세요.
